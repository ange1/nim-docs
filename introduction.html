介绍 ​

概述 ​

Nimblex UI 是一个轻量、易用的 Vue 3 组件库，提供开箱即用的高质量组件，帮助开发者快速构建现代化的 Web 应用。

核心特性：

  • 🚀 基于 Vue 3 Composition API
  • 📦 开箱即用的高质量组件
  • 🎨 简洁优雅的设计风格
  • 💪 完整的 TypeScript 支持
  • ⚡️ 按需引入，优化打包体积

安装 ​

使用你喜欢的包管理器安装：

bash
npm install nimblex-ui
bash
yarn add nimblex-ui
bash
pnpm add nimblex-ui

快速开始 ​

完整引入 ​

在 main.js 中引入所有组件：

js
import { createApp } from 'vue'
import NimblexUI from 'nimblex-ui'
import 'nimblex-ui/dist/style.css'
import App from './App.vue'

createApp(App).use(NimblexUI).mount('#app')

按需引入 ​

只引入需要的组件：

vue
<script setup>
import { NxButton, NxInput } from 'nimblex-ui'
import 'nimblex-ui/dist/style.css'
</script>

<template>
  <NxButton type="primary">按钮</NxButton>
  <NxInput v-model="value" placeholder="请输入" />
</template>

组件示例 ​

按钮组件 ​

vue
<NxButton>默认按钮</NxButton>
<NxButton type="primary">主要按钮</NxButton>
<NxButton type="danger">危险按钮</NxButton>

输入框组件 ​

vue
<NxInput v-model="value" placeholder="请输入内容" />
<NxInput v-model="value" disabled />

表单组件 ​

vue
<NxInput v-model="form.username" placeholder="用户名" />
<NxInput v-model="form.password" type="password" placeholder="密码" />
<NxButton type="primary" @click="submit">提交</NxButton>

VideoPlayer 视频播放器 ​

基于海康H5Player SDK开发的视频播放器组件，支持实时预览、回放、对讲等功能。

视频播放测试 ​

vue
<template>
  <div style="width: 100%; height: 400px">
      <NxInput v-model="url" placeholder="请输入wsurl" :style="'width: 80%; max-width: none;margin-right:10px;'"/><NxButton type="primary" @click="handleClick">播放</NxButton>
     <NxVideoPlayer ref="videoPlayerRef" />
  </div>
</template>

<script setup>
import { ref } from 'vue'
const videoPlayerRef = ref(null);
const url = ref('')
const handleClick = () => {
  videoPlayerRef.value.startPlay(url.value)
}
</script>

Props ​

参数说明类型可选值默认值
id播放器容器IDstring—player-${timestamp}
url播放地址string—''
mode播放模式number0(普通模式) / 1(高级模式)0
defaultSplitNum默认分屏数量number1 / 2 / 3 / 41
autoPlay是否自动播放boolean—true
playbackState是否需要回放功能boolean—false

Events ​

事件名说明回调参数
ready播放器初始化完成(player: JSPlugin)
play-start开始播放(url: string)
window-select窗口选中(iWndIndex: number)
getVideoHistroyData获取视频历史数据(params: object)

Methods ​

通过 ref 可以调用以下方法：

方法名说明参数
startPlay开始播放(url?: string)
play播放视频(data: array)
setSplit设置分屏(num: number)
handlePlaybackData处理历史回放数据(data: object)

历史回放功能 ​

组件支持视频历史回放功能，通过getVideoHistroyData事件获取历史视频数据。

事件参数 ​

getVideoHistroyData事件会触发以下参数：

参数说明类型
deviceId设备通道号string
startTime开始时间string (YYYY-MM-DD HH:mm:ss)
endTime结束时间string (YYYY-MM-DD HH:mm:ss)
flowType流类型string (默认为"ws")

注意事项 ​

  1. SDK自动加载：组件会自动从basePath指定的路径加载H5Player SDK（h5player.min.js），无需手动引入
  2. basePath配置：确保basePath属性指向正确的SDK路径，默认为/H5player/
  3. 播放地址：播放地址需要是有效的海康视频流地址
  4. 生命周期：建议在组件销毁前调用 stopPlay 方法停止播放
  5. 多分屏模式：多分屏模式下，需要确保视频流地址支持多路播放
  6. 高级功能：某些高级功能（如电子放大、智能信息）仅在高级模式（mode=1）下可用
  7. SDK文件：确保将海康H5Player SDK文件（h5player.min.js及相关资源）放置在public/H5player/目录下
